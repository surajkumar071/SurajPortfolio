## 🚀 Quick Start Guide

### 1. **View the Portfolio**
```bash
# Dev server is already running at http://localhost:5173
# Just visit the URL in your browser
```

### 2. **Customize Your Information**
Open `src/App.jsx` and update:
- Your name (Line ~180)
- Your title (Line ~160)
- Your bio (Line ~165)
- Your projects (Line ~540)
- Your skills (Line ~440)
- Your social links (Lines ~185, ~770)

### 3. **Change Colors** (Optional)
Search and replace throughout `src/App.jsx`:
```
red-600 → your-color-600
red-500 → your-color-500
red-400 → your-color-400
```

### 4. **Build for Production**
```bash
npm run build
```

### 5. **Deploy**
- **Vercel**: Push to GitHub and auto-deploy
- **Netlify**: Connect repo and auto-deploy
- **Traditional Hosting**: Upload the `dist/` folder
